Decision Tracker API

This service tracks decisions made via email and provides query capabilities.

Available Endpoints:

POST /api/webhook-inbound

Webhook endpoint for processing incoming emails. Logs decisions when CC'd and responds to queries when directly addressed.

GET /api/setup-decisions-db

Sets up the enhanced decisions database table with LLM processing support. Run this once to initialize the database.

GET /api/decisions-ui

View all confirmed decisions in a user-friendly interface.

GET /api/confirm-decision

Confirmation endpoint for decision approval (accessed via email links).

Status: Enhanced API with LLM-powered decision extraction is deployed and ready!

New: Decisions now require confirmation before being logged. Check your email after CCing the bot.
